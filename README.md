- [x] Fuzzy search
- [x] Multiple themes
- [ ] Song manager (Ability to edit song tags, delete songs)
- [ ] Song downloader with yt-dlp
- [ ] Publish on flathub, the AUR
- [ ] MacOS, Windows support
